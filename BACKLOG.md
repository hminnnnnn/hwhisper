## v0.1.0 완료 (전부 실기기 검증됨)

- **딕테이션 코어**: 우측⌘ 단일 키 토글(우측⌥/fn/조합키 설정 가능), 온디바이스 Apple SpeechTranscriber(한국어 CER 1.9%/0.13s/19MB — 베이크오프 승자), WhisperKit 폴백 구현, 적응형 VAD, 취소 더블탭, 연속 발화 직렬 큐(depth 3)
- **상태머신**: PipelineActor §3.1 정식 통합, 전 상태 전이 로깅
- **삽입**: 클립보드+⌘V 범용 + AX 화이트리스트(read-back 검증), 포커스 변경 시 클립보드 폴백, 보안필드 TOCTOU 거부 — TextEdit/Stickies/Chrome/Safari 실증
- **LLM 정제**: Gemini/Groq/Ollama/커스텀 단일 클라이언트, 다듬기/구조화 2단계(8케이스 매트릭스 검증 — 목록화·정정 제거·단락 분리·한영 혼용), 8s 타임아웃→raw 폴백, 언어 모드(한/영/자동) 설정
- **UI/UX**: 유리 필 인디케이터(웨이브폼·상태 모션·사운드 피드백), 환영 창(권한 상태), 설정 창(단축키/언어/정제/사운드), 메뉴바 상태 아이콘+최근 오류
- **인프라**: hwhisper-dev 고정 서명(TCC 유지), CredentialStore(키체인 프롬프트 지옥 해결), 파일 로그, .app 번들 스크립트, 한국어 README

## 다음 버전(v0.2) 우선순위 — 2026-07-19 사용자 재정렬

1. **독립 앱 셸 + 사용성** ✅ — 메인 창(사이드바: 홈/히스토리/개인 사전/설정, 기본 진입=홈), 먹 다크 고정 브랜드 외형, Dock 승격/복귀. **홈 대시보드** ✅: 이번 주 받아쓰기 시간·단어·절약 시간(40WPM 대비)·WPM, 말하기 버튼 확인/변경, 최근 3건. duration은 v0.2부터 기록(이전 행은 0)
2. **히스토리 + 검색** ✅(1차 구현) — `SQLiteHistoryStore`(LIKE 부분 문자열 검색 — FTS5 unicode61은 한국어 부분일치를 놓쳐 채택 안 함, 0600 파일), raw/정제 쌍 + 대상 앱 + 결과 저장, 복사/삭제/전체삭제/실시간 갱신, 설정 토글. 미검증: 실제 딕테이션→기록 E2E (아래 상시 수집 참조)
3. **개인 사전** ✅(1차 구현) — `FilePersonalDictionary`(JSON 0600, 용어+치환 변형): 인식 바이어싱(contextualStrings) + 정제 보호(protectedTerms 프롬프트) + last-mile 치환(정제 후, 최장 변형 우선·대소문자 무시) 3중 배선 완료. 메인 창 "개인 사전" 탭에서 추가/수정(더블클릭)/삭제. 미검증: 실발화 바이어싱 효과 E2E
4. **언어 자동 감지** — '자동' 모드에서 WhisperKit(단일 다국어 모델) 라우팅 또는 감지 패스. 영어 발화 깨짐 해결. 메모리 트레이드오프는 베이크오프 데이터(B2 RSS 250MB) 참고
5. **음성 명령 모드** — 선택 텍스트에 "더 정중하게" 등 (Wispr Pro 전용 기능의 무료 구현, 정제 LLM 재사용)
6. 스니펫 / 정제 강도 4단계화(Wispr Auto Cleanup 벤치) / 앱별 톤 프리셋
7. 자동 스타일 학습(수정 감지→사전 후보) / 온보딩 위저드 / 사용 통계
8. **iOS** (HwhisperCore 재사용 + 키보드 익스텐션) — macOS v0.2 안정 후
